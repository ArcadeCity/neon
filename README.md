# Neon - Arcade City Wallet

Crypto wallet for the Arcade City ecosystem.

Chains supported:

- Bitcoin
- Solana

Supports:

- Arcade Token
- Bitcoin (Lightning and Solana WBTC)
- USDC

May support in the future:

- Stacks

Auth options:

- Passwordless email via [https://magic.link/](Magic)
- Standard seed phrase
